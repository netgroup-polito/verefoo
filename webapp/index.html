User guide

Load NFFG

  • Download NFFG

    By this, the entire set of NFFG present on the Verigraph server will be downloaded.

  • Load NFFG from file

    Select a file from your local disk as source of NFFG. The file must contains a JSON object.

Save NFFG

  • Save as new NFFG

    Force to save the actual NFFG on the server as new NFFG.

  • Update the NFFG

    Try to update the actual NFFG on the server. (If this graph has been downloaded from the server, the remote NFFG will be replaced by this. If this NFFG hasn't been downloaded from the server, it will be saved as new NFFG.)

  • Save on file

    Save the actual NFFG on a new local file.

Edit graph

  • Add new node

    Add new node to the NFFG (it will be linked with other nodes by creating new edge).

  • Add new edge

  • Edit node

    Change name, functional type and edit configuration of the node.

  • Remove node
  • Remove edge

Create new graph

It simply request to create a new graph, requesting it to the server.

Verify the graph

Your graph will be send to the server in order that will verify it. The graph has not been stored in the server. You will receive an advise if the graph is valid or no or there has been an error during the operation.

Verification graph

Verification of the policy (reachability, isolation, traversal).

View last verification

Show the JSON representation of the last verification.

Remove the graph

Remove the selected graph.

View json data graph

Show the JSON representation of the selected graph.

Modify JSON

Show the JSON representation, modify it by parsing JSON editor (it's not verified by Verigraph rules).
